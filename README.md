# Labtainer iModules

Install the DSSS steganography labs directly with `imodule`:

```bash
imodule https://github.com/bac-ptit/labtainer/raw/main/steg-dsss-embed.tar.gz
imodule https://github.com/bac-ptit/labtainer/raw/main/steg-dsss-noise.tar.gz
```

Run the labs:

```bash
labtainer -r steg-dsss-embed
labtainer -r steg-dsss-noise
```

The repository also includes `steg-dsss-hoangbac2004-imodule.tar`, which contains both labs in one archive:

```bash
imodule https://github.com/bac-ptit/labtainer/raw/main/steg-dsss-hoangbac2004-imodule.tar
```
